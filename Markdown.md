1. Explicação e Justificativa da Solução

A landing page da RE:Direct foi desenvolvida com foco na persona **Carla, 34 anos**, uma profissional de marketing em busca de transição para UX Design. A solução foi pensada para atender seus principais desafios:

- **Insegurança técnica** → Oferecemos trilhas de aprendizado introdutórias e gratuitas.
- **Medo de recomeçar** → Depoimentos reais mostram que a mudança é possível e positiva.
- **Falta de mentoria e comunidade** → Criamos espaços de conexão com profissionais e redes de apoio.

A temática central — **transição de carreira com propósito** — é refletida em toda a estrutura da página, desde o conteúdo emocional até os recursos práticos oferecidos (mentorias, cursos, comunidade).

2. Tecnologias Utilizadas

- React + Vite: Framework moderno e leve para desenvolvimento rápido e eficiente.
- Bootstrap 5: Utilizado para garantir responsividade, componentes acessíveis e layout consistente.
- Vercel: Plataforma de deploy com integração contínua e otimização automática.
- GitHub: Repositório público para versionamento e colaboração.

- **Contraste de cores**: paleta testada com ferramentas como [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) para garantir legibilidade.
- **Texto alternativo em imagens**: todas as imagens possuem `alt` descritivo.
- **Navegação por teclado**: todos os elementos interativos são acessíveis via `Tab`.
- **Responsividade**: layout adaptado para diferentes tamanhos de tela, incluindo leitores móveis.

**Testes realizados:**
- Validação com Lighthouse (Google Chrome DevTools)
- Simulação com NVDA (leitor de tela gratuito)
- Navegação sem mouse para verificar foco e acessibilidade por teclado
